## Commands

- **Dev server**: `npm run dev` (localhost:3000)
- **Build**: `npm run build`
- **Start production**: `npm run start`
- **Lint**: `npm run lint` (ESLint 9 flat config with Next.js core-web-vitals + TypeScript rules)
